10.3 WB PS2 Ramkarte


Speichererweiterung PS/2-SIMM von WB-Systemtechnik




Die PS2 Speichererweiterung von WB-Systemtechnik läuft in allen meinen Falcon F030 bisher Einwandfrei. Die Platine zeichnet sich durch ein inteligentes Routing mit kurzen Leiterbahnen aus. Das geht soweit das diese Karte hingegen anders lautender Aussagen auch korrekt mit 14Mb in einem Nemesis-Falcon bei 48Mhz betrieben werden kann.

Die Karte besitzt einen Jumper mit dem man zwischen einem 4 und einem 16MB PS2-Modul auswählen kann. Bei Lieferung ist der Jumper Default für ein 16MB Simm eingestellt. Achten Sie darauf das sie das PS2-Modul korrekt in den Sockel der Karte einstecken. Es passen auch Module die auf beiden Seiten mit Ramchips bestückt sind.

On this page you will found some intformations about the PS2 Ram expansion of WB-Systemtechnik/Germany. I used this board on one of my falcons well. All programs such like NVDI, MagiC and any other applications run well in 14MB of ST-Ram.

The board use a inteligent track routing with real short tracks. That follows that i can use this board also well with an Nemesis and an CENTurbo2 Falcon. Nemesis Falcon runs well with 48Mhz CPU clock.

On this board you will found one jumper they change the memory size from 4 to 14MB. 16 MB PS2 Simm loose 2 MB of RAM, that's normal and it is a Falcon memory design problem. Watch if the module is pluged well on memory board. Do not touch Simm connectors with your fingers. You got the best results with 60nS memory modules.

I used an PS2 Simm from Hundai GT-7233T well.

If youo doans't have 60nS modules, vertical stripes, dots comes up if the Falcon is accellerated.

Plug in memory board to Falcon internal memory port. Watch that the board is connected well to both rows of the memory expansion port. If the board is pluged wrong, memory board and Falcon board goes damaged.
