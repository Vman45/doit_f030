18.15 TeleInfo Atari


TeleInfo - Telefonnnummern für den Atari

Die Tele-Info-CD Atari enthält alle vom Tele-Info Verlag zum Ausgabedatum veröffentlichten Einträge von Geschäfts- und Privatadressen sowie die dazugehörigen Telefonnummern. Insgesamt befinden sich laut Hersteller auf dieser CD ca. 

- 35 Milionen Telefonanschlüsse privat/gewerblich
-  1 Million Faxnummern
-  5 Millionen Branchen und Berufe
-  120.000 T-Online Nummern

Die TeleInfo-CD erfordert einen Atari ST/TT/Falcon mit CD-ROM Drive und ca. 2 MB freien Arbeitsspeicher. Das TeleInfo-Programm liegt auf meiner neuen CD in der Version 1.2 vor.
Das TeleInfo-Suchprogramm läuft bei mir ohne Probleme unter TOS 4.04 oder aber MagiC 5.10 auf dem Falcon.


Abbildung 1 - Frontcover der älteren TeleInfo CD Atari

Kopiert man den Ordner "DATA" und die .IND-Files aud seine Festplatte hat man einen wesentlich schnelleren Zugriff auf die Telefonnummern, auch die Suche läuft schneller ab.
Man sollte zu diesem Zweck gut 30Mb freien Plattenplatz abzweigen können. Ich halte die Installation der Daten auf einer Festplatte überflüssig, es gib genügend schnelle CD-Drives.

Ansonsten genügt das anlegen eines Ordners "Teleinfo" Hier kopiert man das TELEINFO.PRG und sein TELEINFO.RSC hinein.
Die CD liegt im CD-ROM und wird automatisch gefunden, das geschieht auch wenn mam mehr als ein CD-ROM an seinem Atari angeschlossen hat.

Die Suche kann nach Namen, Nachnamen, Straßen- und Ortsnamen erfolgen. Das suchen nach Telefonnummern ist aus Datenschutz-rechtlichen gründen nicht implementiert.
Die Suche ansich läuft selbst wenn die Daten von einem 2-fach Speed Drive geladen werden in einer brauchbaren Geschwindigkeit ab.

Die Teleinfo CD Atari kann bei der Fa. R.O.M-logicware in Berlin oder bei delta labs in Wuppertal bezogen werden.

Alles in allem ist so eine schnelle Suche von Teilnehmern nun auch komfortabel an unseren Atari-Computern möglich.
Die TeleInfo CD ist mit 623MB recht gut gefüllt.

Bleibt noch zu sagen das auch die zweite Ausgabe der TeleInfo CD auf meinem System einwandfrei arbeitet, diese zweite Ausgabe ist eine Mac/Atari-Hybrid CD-Rom. Ebenfalls gibt es auf meinem Mac keine grösseren Probleme damit. Negativ aufgefallen ist die Tatsache das lediglich auf der Rückseite der CD Hülle in sehr kleiner Schrift vermerkt wurde das diese CD 'auch' auf dem Atari benutzt werden kann.
