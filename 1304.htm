19.4 Videlity


Videlity ScreenEnhancer Software


Mit 'Videlity' päsentiert seit einiger Zeit die Firma Titan Design eine Bildschirmerweiterungs-Software die u.A auch mit den hier bekannten Hardware Erweiterungen wie ScreenBlaster oder BlowUp problemlos zusammen arbeitet

Videlity ist ein Screen Enhancer der den Bildschirm eines Atari Falcon F030 schnell und problemlos auf eine höhere Auflösung bringt. Die Videlity-Software ersetzt hierbei die Sofware welche Sie mit dem ScreenBlaster oder BlowUp erhalten haben vollständig. Ebenso wie in der hier bereits bekannten VMG-Software ist es in Videlity möglich einen Schutzmechanismus für den verwendeten Monitor zu aktivieren. Das ist auch ratsam, betrachtet man die einstellbaren Auflösungen.

Die Installation von Videlity beschränkt sich auf das eingeben des Names, seiner eigenen Adresse. Die Disketten sind einzeln seriealisiert und besitzen bei der ersten Benutzung nur das Registrierungsprogramm. Ihre persönlichen Daten sollten Sie hier peinlichst genau eingeben, Sie haben nur einen einzigsten Versuch.


Abbildung 1 - Videlity Schnappschuß



Nach dem Beenden der Registrierung wird der Rechner einen Kaltstart durchführen
Die Installation kann nur auf der Videlity Diskette erfolgen
Nach dem Kaltstart finden Sie alle nötigen Dateien auf der Diskette. 

- VIDELITY.PRG, das Programm für den Autoordner
- VID_CONF.PRG, den 'Videomode Generator'
- VID-CONF.INF, das Setup dazu
- VIDELITY.HYP, die englsiche Anleitung als Hypertext
- STD_CONF.RGB, Standardkonfiguration für verschiedene RGB-Monitore
- STD_CONG.VGA. Standardkonfiguration für verschiedene VGA-Monitore

Die meisten der Einstellungen im VMG werden über Slider vorgenommen. Man kann hier so sehr leicht jeden einzelnen Parameter rasch verändern. Zum Testen der Bildschirmauflösung ist ein großer 'Test'-Button vorhanden.

Betätigen Sie den Button für die externe Hardware und es ist eine Hardware bereits Angeschlossen, taucht ein Menü auf in dem Sie sicht zwischen der ScreenBlaster- oder der BlowUp-Hardware entscheiden müssen.

Beachten Sie bitte das alle Einstellungen im VMG sofort wirksam werden
Veränderte Settings können jederzeit gespeichert werden

Zum Anpassen der Sicherheitseinrichtung sollten Sie unbedingt die wichtigsten Eckdaten ihrer Monitoranleitung entnehmen und im VID_CONF.PRG korrekt eintragen. Nur so kann Gewährleistet werden das der von ihnen verwendete Monitor nicht beschädigt wird.

Besonders hohe Bildschirmeinstellungen sind möglich wenn sich ein Nemesis Beschleuniger im Falcon befindet. Hier sind z.B unter TrueColor Auflösungen bis zu 640x480 Non-Interlaced leicht möglich
Die Anschrift- und Emailadresse von Titan Design finden sie unter Händler



Videlity is a Screen enhancer for any Atari Falcon 030 computer system enabling increased resolutions for the Atari System. The program also provides support for users of the external hardware supplied with the BlowUp and ScreenBlaster Hardware, and is a complete replacement for the software provided with these products.This product was designed to provide access to the high resolution true color VGA modes which Nemesis makes possible.

Copy VIDELITY.PRG to your auto folder and make sure that it is placed after NVDI (if installed) in the running order. Now disable any other screen enhancement software you may have installed and reboot your machine. Use the VMG to set your own Screen resolution.
