8.1 Falcon Videoadapter


Falcon Videoschnittstelle


Wenn Sie sich einen Videoadapter, abhängig vom verwendeten Monitor, selberbauen müssen, sollten Sie einige grundlegende Dinge beachten. Sehr wichtig ist das Sie dem Computer, dessen Videologik, mitteilen welcher Monitor wirklich angeschlossen ist. Beachten Sie das nicht, kann Monitor und Videohardware im Computer, ernsthaft beschädigt werden. Der 19 polige Falcon Videoanschluß hat aus diesem Grund den Pin 18 (M1) und den Pin 19 (M0). Beide Pin dienen als eine Art Auswahljumper für den richtigen Betriebsmodus.
Beide Pin werden bei Benutzung ausschliesslich mit GND verschaltet!

Wie das funktionieren entnehmen Sie aus der unten folgenden Tabelle.


M1 M0 Monitor type
0 0 ST Monochrome
0 1 ST Color
1 0 VGA
1 1 TV



Die Pinbelegung, die einzelen Signale, finden Sie in der nächsten Tabelle. Auch hier ist Pin 18 und Pin 19 mit aufgeführt. Die nachfolgende Tabelle zeigt eine komplette verschaltung für einen handelsüblichen VGA-Monitor. Unbenutze Pin sind nicht aufgeführt.

Wie Sie hier erkennen können ist die Falcon Videoschnittstelle gar nicht so weit, bis auf einige Sonderpins und einen 19 poligen Stecker, von einem 15 Pin HD Stecker entfernt. Die Videosignale ansich entsprechen mit ihrem Signalpegeln dem allgemeinen Standard.

Alle GND Pin können mit einer Drahtbrücke verbunden werden.
(Billigkabel)

Die beste Bildqualität, beim anfertigen von Adaptern, erhalten Sie immer wenn hochwertige Industriekabel verwendet werden. Grundsätzlich wird, auch für einen Videoadapter, immer abgeschirmte Leitung verwendet. Höherwertige Kabel verweden für die Signale R,G,B und die beiden Syncronisationssignale HSYNC und VSYNC jeweils eine einzeln abgeschirmte Ader im Kabel. Das vermindert Signalübersprechen und minimiert Störungen durch äussere Einflüsse.

Wie Sie sicher schon gesehen haben gibt es daher auch die GND Pin die jeweils zu Rot, Grün und Blau passen. Die wären dann die richtigen GND Pin für die jeweilige Einzelabschirmung. Ausschliesslich so erhält man die höchste Bildqualität. Es nutzt allerdings wenig wenn Sie ihren Adapter so löten, der Hersteller ihres Monitor aber ein Billigheimer ist, denn dann ist das original Monitorkabel garantiert so nicht aufgebaut.
