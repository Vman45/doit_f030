2.8 CD Service


CD Service



Der geneigte Leser kann sich bei mir immer die aktuellsten Archive auf einer CD-R bestellen, lange bevor man sich das jeweilige Archiv im Internet Downloaden kann.



Postversand

Generell funktioniert dieser Service nur wenn Sie einen an sich adressierten und frankierten Briefumschlag inklusive dem gewünschen Medium einsenden.

Sendunden ohne freien Rückumschlag, ohne Porto, ohne Adresse werden nicht bearbeitet. Einsendungen die mit Nachporto belegt sind werden nicht angenommen oder direkt vernichtet.

Postversand erfolgt generell in der Woche des Posteingangs.



Die Kosten

Generell keine, DOIT-Archive sind -=FairWare=-

Sollten Sie keine Medien besitzen, können Medien von mir bezogen werden.
Eine bespielte CD-R kostet 5 Euro zuzügl. Porto u. Versand.
Ein bespieltes ZIP-Medium kostet 5 Euro zuzügl. Porto u. Versand.



Datenträger

CDs können CD-R oder CDRW sein. Das liegt nur an ihnen. Format ohne Angabe wird immer ISO6996 sein. Wünschen sie Atari oder Mac HSF Format sollten sie das gesondert angeben. Pure DOS/Windows CDs werde ich nicht erstellen. Alles das trifft auch auf ein ZIP-Medium zu. ZIP Medien können Sie Atari kompatibel oder im Mac HSF- Format erhalten. Ohne spezielle Angabe wird ein ZIP-Medium immer Atari kompatibel formatiert sein. Sie sparen mir Arbeit wenn alle eingesendeten Medien, bis auf CD-R, schon so formatiert sind wie sie es selbst gerne hätten.

Defekte Datenträger werden von mir nicht ersetzt.
