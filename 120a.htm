    Konzept
    -------

Bei  CDMENU  handelt  es   sich   um  eine  einfache  fenstergestützte
Oberfläche für CDs.  Eine  CD  enthält  genauso  wie  eine Platte eine
Hierarchie von Ordnern. Irgendwo unten in der Hierarchie befinden sich
Ordner, die nur  noch  zusammengehörige  Dateien  enthalten, z. B. ein
Programmpaket oder eine Bildersammlung  o.  ä. Ein solcher Ordner wird
Paket genannt.

Die oberste Verzeichnisebene wird  in  einer Dialogbox mit Buttons für
die einzelnen  Kategorien  angezeigt.  Die  weiteren Ordner erscheinen
dann in Iconfenstern an, wie sie  vom Desktop her bekannt sind. Sobald
sich der Anwender aber bis  auf  die Ebene eines Paketes durchgeklickt
hat, erhält er ein Infofenster.  Dieses  enthält einen Infotext zu dem
fraglichen Paket und bietet einige Operationen  an. So ist es möglich,
ein Programm direkt von  der  CD  zu  starten,  gepackte Daten auf die
Festplatte des  Anwenders  auszupacken,  Textdateien  anzusehen  (oder
Sounddateien anzuhören!), sowie Archive oder  einzelne Dateien auf die
Festplatte  des  Anwenders  zu   kopieren.   Dazu   stehen  Ihnen  die
eingebundenen Programme 1st  GUIDE  und  diverse (Ent-)Packerprogramme
zur Verfügung.
Drag & Drop ist in  Zusammenarbeit  mit  einem AV-Server (z.B. GEMINI)
unter MTOS oder MagiC möglich. Andernfalls wird, wenn ein oder mehrere
Ordner aus einem Fenster gezogen werden, ein Fileselector geöffnet, um
das Kopierziel abzufragen.

    Installation
    ------------

CDMENU.APP finden  Sie  im  Wurzelverzeichnis  der  CD;  alle  anderen
Dateien in einem  Ordner  namens  CDMENU  (den  Sie  als Anwender aber
getrost vernachlässigen können). Falls Sie den Inhalt der vorliegenden
CD  via  CDMENU  sondieren  wollen,  starten  sie  dazu  das  Programm
CDMENU.APP  und  wählen   Sie   dann   aus   dem   Button-Fenster  die
entsprechende Kategorie aus.  Danach  können  Sie  das  jeweilige Icon
öffnen  und  erhalten   dann   ein   entsprechendes  Info-Fenster  mit
Programminformationen  und   den   Start-,   Kopier-,   Anzeige-   und
Entpackbuttons. Sollten Sie die  Warn-Meldung  "Kann  den Ordner nicht
öffnen" erhalten, befinden Sie sich in einem Subverzeichnis, das keine
anwenderrelevante  Dateien  enthält,  so   daß  sie  dieses  Operation
"canceln" können.
