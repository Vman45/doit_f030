CENTurbo2 - Modifikation der 2.5" IDE Halterung


Es folgt eine kleine Umbauanleitung für die IDE-Halterung der 2.5" Festplatten im Atari Falcon. Wenn Sie einen CENTurbo2-Beschleuniger in ihren Falcon eingebaut haben werden, Sie schnell feststellen, daß man die interne 2.5" Festplatte nicht mehr wie gewohnt befestigen kann. Nachfolgend sehen Sie hier einige Bilder, die meine eigene CT2 Installation und meine Idee zum Modifizieren des internen 2.5" Laufwerkträges beinhalten. Die interne 2.5" IDE Festplatte befindet sich bei diesem Einbau fast an dem orignalen Platz, allerdings in Längsrichtung zwischen dem Diskettenlaufwerk und dem internen Netzteil.

Das vorhandene IDE-Kabel kann in diesem Fall weiter verwendet werden.

Zum Umbauen benötigen Sie eine Eisensäge und eine mittlere Flachfeile sowie einen Schraubstock. Dinge, die nicht jeder hat, ohne die es aber leider nicht geht.

Wie Sie unschwer aus den Bildern entnehmen können wurden die hinternen Beine des Laufwerksträgers von mir abgetrennt, alle Sägekanten mit der Feile begradigt und die Späne sorgsam entfernt. Danach wird der so zerlegte Laufwerksträger -nur dessen Vorderteil- einfach quer zur normalen Einbaurichtung mit nur zwei Schrauben in den beiden linken Schraubenlöchern auf der Hauptplatine erneut angeschraubt. Es sind zwar nur zwei Schrauben, immerhin ist diese Sache aber insgesamt stabiler als eine lose hineingelegte 3.5" Festplatte.













Abbildung 1-5 Modifikation des 2.5" Laufwerkträger



Eine vorhandene 2.5" Festplatte kann auf dem Laufwerksträger wie gewohnt befestigt werden. Wenn es in Richtung des Netzteiles eng werden sollte, bohren Sie einfach neue Befestigungslöcher für die Festplatte in den Träger.

Achtung:

Beachten Sie bitte dringlich, daß die Metallbeine des Laufwerkträgers auf der Hautplatine keine Leiterbahnen berühren! Schneiden Sie kleine Scheiben aus Pappe aus und legen Sie diese unter die Befestigungsbeine, bevor Sie diese anschrauben. Vergessen Sie niemals die Isolierscheiben! (Pappe)

Denken Sie bitte daran, daß Bohr- oder Feilspäne den Computer beschädigen können!


Befestigung einer 3.5" Festplatte

Hier finden Sie eine Möglichkeit zum Befestigen einer 3.5" Festplatte, zumindest mit einer einzigen Schraube. Für diese Methode muß ein Loch in das Gehäuse gebohrt werden. Sie müssen die Festplatte so hinrücken, daß Sie passt, um dann die Bohrstelle markieren zu können.
