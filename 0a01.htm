10.1 Falcon Ramkarten


Speicherkarten im Atari Falcon



Im Atari Falcon F030 kommen bisher zwei verschiedene Speicherkarten
zum Einsatz, es gibt hier eine Originale 4 MB und eine 16MB Karte
von der allerdings nur 14MB nutzbar sind.

Beide Karten werden auf den internen Memoryport des Falcon gesteckt.
Die Karten sollten niemals um einen oder mehrere Pins versetzt aufgesteckt werden. Schäden am Computer und an der Karte können so leicht verhindert werden. Grundsätzlich werden die Karten auch daher eigentlich nie ausgebaut oder gar im Betrieb abgezogen!

Die Pinbelegung der beiden Stiftleisten der Speichererweiterung.

Die Speicherkonfiguration sieht vor das für 1MB RAM die Pins 23 und 48 auf GND gelegt sein müssen damit das RAM korrekt erkannt wird. Pin 23 wird bei einer 4MB Karte auf GND gelegt, Pin 48 bleibt hingegen offen. Ist nur Pin 48 gegen GND gelegt sollte eine 14MB-Karte auf dem Memoryport gesteckt sein. Die PS/2-Speicherkarten von WB-Systemtechnik haben zu diesem Zweck den Pin 23 und 48 (Memory Size 0 und Memory Size 1) auf einen Jumper gelegt. Man erhält hier die Möglichkeit ein 4MB oder ein 16MB Modul einzustecken.



Tabelle 15: Memory Jumper
Pin 23 Pin 48 Size
x x = 1MB ST Ram
x = 4MB ST Ram
x = 14MB ST Ram



Legend:
x = Jumper gegen GND

Bleibt noch zu sagen das Pin 48 (DRAM1) des Memoryport über Pin 7 (R7) des Pulluparrays (P6) mit Pin 13 von U42 (74LS244) verbunden ist. Ebenso findet die Leitung von Pin 23 des Memoryport ihren Weg über Pin 6 (R6) des Widerstandsarrays zu Pin 11 (DRAM0) von U42. U42 selber dürfte ihnen schon aus dem Videoteil her bekannt sein.

So wie im Videoteil hier erkannt wird welcher Monitor am Falcon angesteckt ist, wird hier ebenso erkannt wieviel RAM so am Memoryport eingesteckt wurde. DRAM0 und DRAM1 sind die beidem Selektleitungen die man untersuchen sollte wenn z.B das Ram nicht korrekt erkannt wird.
Die Pullups von P6 liegen an +5Vcc
Sämtliche Datenleitungen der beiden Stiftleisten liegen als Bus direkt an wichtigen Custom-Bausteinen an. Das Betrifft MDATA0-MDATA31 sowie MADDR0-MADDR10 und ebenfalls sämtliche CAS- und RAS-Leitungen.
Diese Leitungen werden von der im COMBEL enthaltenen MMU im Multiplex-Betrieb bedient. Kurzschlüsse sind zu vermeiden!
Seltsam erscheint die Nummernreihenfolge der Signalleitungen an beiden Stiftleisten, es ist in den Schaltplänen aber ebenso wie im Servicemanual nummeriert. Man kann es anhand der Lage beider Memory-Selektleitungen leicht kontrollieren.
